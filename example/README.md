# Examples

Go to examples folder and run php build in server
```php
php -S 127.0.0.1:8081
```

Open 127.0.0.1:8081 in your browser.

Examples are provided for: 
⋅⋅* Credit check;
⋅⋅* Request order;
⋅⋅* Request Status order;
⋅⋅* Start workflow;
⋅⋅* Add order credit;
⋅⋅* Add order payment;
⋅⋅* Hold order;
⋅⋅* Resume order;
⋅⋅* Order file (invoice pdf);
⋅⋅* Add message to order.
